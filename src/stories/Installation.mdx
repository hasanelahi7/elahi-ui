# Installation

Get started with elahi-ui in your React project.

## Install Package

```bash
npm install elahi-ui
```

**Peer Dependencies** (required):
```bash
npm install react react-dom
```

## Setup Tailwind CSS

### 1. Install Tailwind

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### 2. Configure Tailwind

Update your `tailwind.config.js`:

```js
export default {
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/elahi-ui/dist/**/*.{js,mjs}",
  ],
  darkMode: 'class',
  theme: {
    extend: {},
  },
  plugins: [],
}
```

### 3. Import elahi-ui Styles

Import the theme CSS variables in your main CSS file or entry point:

```tsx
// In your main.tsx or App.tsx
import 'elahi-ui/styles'
```

**Or** add to your main CSS file:

```css
@import 'elahi-ui/styles';

@tailwind base;
@tailwind components;
@tailwind utilities;
```

## Usage

```tsx
import { Button, Card, CardHeader, CardTitle, CardContent } from 'elahi-ui'

function App() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Welcome</CardTitle>
      </CardHeader>
      <CardContent>
        <Button variant="primary">Get Started</Button>
      </CardContent>
    </Card>
  )
}
```

## Dark Mode

Enable dark mode by adding the `dark` class:

```tsx
<html className="dark">
  <body>...</body>
</html>
```

## TypeScript

elahi-ui is written in TypeScript and includes full type definitions:

```tsx
import type { ButtonProps } from 'elahi-ui'

const MyButton: React.FC<ButtonProps> = (props) => {
  return <Button {...props} />
}
```

That's it! Browse the components in the sidebar to get started.
