# elahi-ui Components

## 14 Production-Ready Components (100% Tailwind CSS)

### Foundational (3)
- ✅ **Button** - Multiple variants, sizes, loading states, ripple effects
- ✅ **Input** - Floating labels, validation, prefix/suffix icons
- ✅ **Textarea** - Auto-resize, character counter

### Typography (2)
- ✅ **Heading** - Responsive headings with gradient option
- ✅ **Text** - Body text with truncation

### Form Components (3)
- ✅ **Checkbox** - Custom styled checkbox
- ✅ **Switch** - Toggle switch
- ✅ **Select** - Dropdown with search functionality

### Interactive (5)
- ✅ **Modal** - Dialog with focus trap and ESC handling
- ✅ **Toast** - Notification system with provider
- ✅ **Tabs** - Tabbed interface
- ✅ **Tooltip** - Hover tooltips with positioning

### Data Display (3)
- ✅ **Card** - Content cards with compound components
- ✅ **Badge** - Status indicators
- ✅ **Avatar** - User avatars with status dots

## Bundle Size
- **ESM:** 51KB (10.6KB gzipped)
- **CJS:** 29KB (8.2KB gzipped)
- **Zero third-party UI libraries**
